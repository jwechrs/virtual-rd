VRD (Virtual Remote Desktop)

Download from SideQuest

VR desktop for VNC users. With VNC server of your choice, you can see the screen, control the pointer and page scrolling. This application is tested on Oculus go with the following VNC servers:

  • TigerVNC (version 1.7.0) on Ubuntu 18.04
  • macOS Build-in screen sharing (setting app>sharing>screen sharing)
  • TightVNC (version 2.8.27) on Windows 10

Release note (v0.2.1)

  • Support new encoding (Hextile)
  • Support page scrolling

Usage

  • Download apk from Sidequest or from Github
  • Install the application to Oculus device by running "adb install /path/to/build.apk"
  • Start VNC server on your PC
  • From Oculus device, go to "unknown sources" page and start VRD
  • Access your host name / port of a VNC server

Protocol Specifications

This app uses RFB protocol with the following specifications.

RFB Protocol version requested by this App003.003
Supported encodingsRAW, Hextile
Supported color depth32bits
