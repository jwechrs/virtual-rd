VRD (Virtual Remote Desktop)

VR desktop for VNC users. This application is tested on Oculus go with the following VNC servers:

▪ TigerVNC (version 1.7.0) on Ubuntu 18.04

▪ macOS Build-in screen sharing (Setting app>Sharing>screen Sharing)

▪ TightVNC (version 2.8.27) on Windows 10

Usage

▪ Download apk from Sidequest or from Github

▪ Install the application to Oculus device by running "adb install /path/to/build.apk"

▪ Start VNC server on your PC

▪ From Oculus device, go to "unknown sources" page and start VRD

▪ Access your host name / port of a VNC server
